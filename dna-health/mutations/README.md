## Understanding the Notation

- **+/+** (Homozygous risk): Both chromosomes carry the risk variant — strongest effect
- **+/-** (Heterozygous): One risk copy — moderate effect, still clinically relevant
- **Confirmed**: rsID found in AncestryDNA raw data and genotype verified
